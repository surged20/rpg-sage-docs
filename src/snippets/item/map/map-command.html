Map Command

/sage map addimage

This slash command is used to add images to the map.

The following options are required:

  • map : Map id (Map post message id)
  • layer : one of aura, terrain, token
  • url : image url
  • name : image name
  • cols : image width
  • rows : image height
  • col : image column position
  • row : image row position

/sage map addimage map:1156918903881093130 layer:token url:https://rpgsage.io/images/docs/map/example/Hoobla.png name:Hoobla cols:1 rows:1 col:12 row:4
Adds a token to the map.
