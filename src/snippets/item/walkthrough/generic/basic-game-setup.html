Basic Game Setup

Send these commands by posting them to the my-game-gm channel.
make sure you are in the #my-game-gm channel.

Create a game called: "My Game".
    sage!!game create name="My Game"

This example has Prefix sage, Action !!, Command game, SubCommand create, and Argument name with a value of My Game.

If you are playing a specific game (such as Pathfinder or Starfinder) or system (such as Essence 20), you may want to access features specific to them (such as dice). Essence 20 refers to the underlying game engine used by Renegade Game Studios' G.I. Joe, Power Rangers, and Transformers games.

Update the game's type optional step.
    sage!!game set type="PF2E"
Acceptable type values are:
    DND5E DnD 5e, E20 Essence 20, PF1E Pathfinder 1e, PF2E Pathfinder 2e, QUEST Quest, and SF Starfinder.

This example has Prefix sage, Action !!, Command game, SubCommand set, and Argument type with a value of PF2E.

Update the channel type to Game Master, which enables all commands for game masters while ignoring players. (RPG Sage assumes players don't have access to this channel.)
    sage!!channel set type="GM"

This example has Prefix sage, Action !!, Command channel, SubCommand set, and Argument typewith a value of GM.

Add the In Character channel to the game, allowing game masters and players to use dialog and dice.
    sage!!channel add #my-game-ic type="IC"

This example has Prefix sage, Action !!, Command channel, SubCommand add, and Arguments #my-game-ic and type=IC.

Add the Out of Character channel to the game, allowing game masters and players access to all of the features.
    sage!!channel add #my-game-ooc type="OOC"

This example has Prefix sage, Action !!, Command channel, SubCommand add, and Arguments #my-game-ooc and type=OOC.

Add the Game Master(s) to the game. You can have as many or as few game masters as you want.
    sage!!gm add @GmMention

This example has Prefix sage, Action !!, Command gm, SubCommand add, and Argument @UserMention.

Add the Players to the game. You can have as many or as few players as you want.
    sage!!player add @Player1Mention @Player2Mention @Player3Mention @Player4Mention

This example has Prefix sage, Action !!, Command player, SubCommand add, and Arguments @Player1Mention, @Player2Mention, @Player3Mention, @Player4Mention.
